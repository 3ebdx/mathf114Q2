Équations Différentielles

Chapitre 1 · Méthodes calculatoires · Exam 2h

FONDAMENTAUX

Qu'est-ce qu'une EDO ?

Définition

Une équation différentielle ordinaire (EDO) est une équation dont l'inconnue est une fonction $y(x)$ et qui contient ses dérivées $y', y'', \ldots$

Ordre : ordre de la dérivée la plus élevée présente.

$$F(x, y, y', \ldots, y^{(n)}) = 0$$
Problème de Cauchy

Résoudre une EDO avec une condition initiale $y(x_0) = y_0$ (ou $y(x_0)=y_0$, $y'(x_0)=y'_0$ pour l'ordre 2).

⇒ La condition initiale détermine la constante $C$ et donne une seule solution à la place d'une infinité.

Les 4 types à maîtriser pour l'exam :

Variables Séparables

Type A · Ordre 1

THÉORIE

EDO à Variables Séparables

Forme
$$y' = g(x)\cdot h(y) \qquad \text{ou} \qquad f(y)\,dy = g(x)\,dx$$

Reconnaître : le membre de droite est un produit de fonctions séparées de $x$ et de $y$.

Méthode — 4 étapes
  1. Séparer : $\displaystyle\frac{dy}{h(y)} = g(x)\,dx$
  2. Intégrer des 2 côtés : $\displaystyle\int \frac{1}{h(y)}\,dy = \int g(x)\,dx$
  3. Ajouter la constante $+C$ (d'un seul côté)
  4. Isoler $y$ si possible — sinon laisser implicite
⚠️ Cas particuliers : Si $h(y_0)=0$, alors $y = y_0$ est aussi une solution (solution constante/singulière). Ne pas oublier de vérifier.
Exemple résolu pas à pas
Résoudre : $y'\sqrt{1-x^2} + xy = 0$
1
Réécrire et séparer
$\dfrac{dy}{dx}\sqrt{1-x^2} = -xy \;\Rightarrow\; \dfrac{dy}{y} = \dfrac{-x}{\sqrt{1-x^2}}\,dx$
2
Intégrer des 2 côtés
$\int\dfrac{1}{y}\,dy = \int\dfrac{-x}{\sqrt{1-x^2}}\,dx$
$\ln|y| = \sqrt{1-x^2} + C$
Astuce : $\int \frac{-x}{\sqrt{1-x^2}}dx = \sqrt{1-x^2}$ (substitution $u=1-x^2$)
3
Isoler y
$|y| = e^{\sqrt{1-x^2}+C} = e^C \cdot e^{\sqrt{1-x^2}}$
$\boxed{y = K\,e^{\sqrt{1-x^2}}}$ avec $K\in\mathbb{R}\setminus\{0\}$
(et $y=0$ est aussi solution singulière, incluse pour $K=0$)
Exemple avec condition initiale (Cauchy)
Résoudre : $(1+y)\,dx - (1+x)\,dy = 0$, avec $y(1) = 2$
1
Séparer
$\dfrac{dy}{1+y} = \dfrac{dx}{1+x}$
2
Intégrer
$\ln|1+y| = \ln|1+x| + C$
$|1+y| = e^C|1+x| \;\Rightarrow\; 1+y = K(1+x)$
$y = K(1+x) - 1$
3
Appliquer la condition initiale $y(1)=2$
$2 = K(1+1) - 1 \;\Rightarrow\; 2 = 2K-1 \;\Rightarrow\; K = \tfrac{3}{2}$
$\boxed{y = \dfrac{3}{2}(1+x) - 1}$
EX-SEP-1 Variables séparables
Résoudre : $(1-y^2)\,dy - y\,dx = 0$
$\int\frac{1-y^2}{y}dy = \int(\frac{1}{y} - y)\,dy = \ln|y| - \frac{y^2}{2}$
EX-SEP-2 Séparable + Cauchy
Résoudre avec $y(0) = 1$ : $y' = (1+y^2)x^2$
$y'=\frac{dy}{dx}$ donc $\frac{dy}{1+y^2} = x^2\,dx$

Équations Linéaires d'Ordre 1

Type B · Ordre 1

THÉORIE

Méthode du Facteur Intégrant

Forme standard
$$y' + p(x)\,y = q(x)$$

Si $q(x)=0$ : équation homogène. Sinon : équation complète.

Méthode — Facteur intégrant
  1. Calculer $\mu(x) = e^{\int p(x)\,dx}$
  2. Multiplier les deux membres par $\mu(x)$ :
    $[\mu(x)\,y]' = \mu(x)\,q(x)$
  3. Intégrer des deux côtés :
    $\mu(x)\,y = \int \mu(x)\,q(x)\,dx + C$
  4. Diviser par $\mu(x)$ pour isoler $y$
⚠️ La prof utilise aussi la variation de la constante : trouver $y_h$ (sol. homogène), puis poser $y = v(x)\cdot y_h$ et trouver $v(x)$ par substitution. Les deux méthodes donnent le même résultat — choisir celle qu'on maîtrise le mieux.
Exemple résolu pas à pas
Résoudre : $y' + 2xy = 4x$
1
Identifier $p(x)$ et $q(x)$
Forme standard : $y' + \underbrace{2x}_{p(x)}y = \underbrace{4x}_{q(x)}$
2
Résoudre l'homogène $y' + 2xy = 0$
$\dfrac{dy}{y} = -2x\,dx \;\Rightarrow\; \ln|y| = -x^2 \;\Rightarrow\; y_h = Ce^{-x^2}$
3
Chercher solution particulière : poser $y_p = k$ (constante)
$y_p' = 0$, donc $0 + 2x\cdot k = 4x \;\Rightarrow\; k=2$
$y_p = 2$
4
Solution générale
$\boxed{y = Ce^{-x^2} + 2}$
Exemple avec CI — Facteur intégrant
Résoudre : $y' + y\cot x = 5e^{\cos x}$, avec $y(\pi/2) = -4$
1
Facteur intégrant
$\mu = e^{\int \cot x\,dx} = e^{\ln|\sin x|} = \sin x$
2
Multiplier par $\sin x$
$(\sin x\cdot y)' = 5e^{\cos x}\sin x$
3
Intégrer à droite
$\int 5e^{\cos x}\sin x\,dx = -5e^{\cos x} + C$
(sub. $u=\cos x$, $du=-\sin x\,dx$)
$\sin x\cdot y = -5e^{\cos x} + C \;\Rightarrow\; y = \dfrac{C - 5e^{\cos x}}{\sin x}$
4
Condition initiale $y(\pi/2) = -4$
$-4 = \dfrac{C - 5e^0}{1} = C-5 \;\Rightarrow\; C=1$
$\boxed{y = \dfrac{1-5e^{\cos x}}{\sin x}}$
EX-LIN-1 Linéaire O1
Résoudre : $(x-2)y' = y + 2(x-2)^3$
Diviser par $(x-2)$ : $y' - \dfrac{1}{x-2}y = 2(x-2)^2$. Donc $p(x) = -\dfrac{1}{x-2}$

EDO Ordre 2 — Homogène C.C.

Type C · Ordre 2 · Coefficients Constants

THÉORIE

Équation caractéristique

Forme
$$ay'' + by' + cy = 0 \qquad (a,b,c \in \mathbb{R},\; a\neq 0)$$
Méthode
  1. Poser $y = e^{\lambda x}$ et substituer : obtenir $a\lambda^2 + b\lambda + c = 0$
  2. Calculer $\Delta = b^2 - 4ac$
  3. 3 cas selon $\Delta$ :
CasRacinesSolution générale
$\Delta > 0$ $\lambda_1, \lambda_2$ réelles distinctes $y = C_1 e^{\lambda_1 x} + C_2 e^{\lambda_2 x}$
$\Delta = 0$ $\lambda$ double $= -b/2a$ $y = (C_1 + C_2 x)e^{\lambda x}$
$\Delta < 0$ $\alpha \pm i\beta$ complexes $y = e^{\alpha x}(C_1\cos\beta x + C_2\sin\beta x)$
⚠️ Pour $\Delta < 0$ : $\lambda = \alpha \pm i\beta$ avec $\alpha = -b/2a$ et $\beta = \sqrt{-\Delta}/2a$. Ne jamais écrire $Ce^{i\beta x}$ dans la solution finale — toujours développer en $\cos/\sin$.
Les 3 cas — Exemples
Δ>0
2 racines réelles — $y'' - 4y' + 3y = 0$
$\lambda^2 - 4\lambda + 3 = 0 \;\Rightarrow\; (\lambda-1)(\lambda-3) = 0$
$\lambda_1 = 1,\; \lambda_2 = 3$
$\boxed{y = C_1 e^x + C_2 e^{3x}}$
Δ=0
Racine double — $y'' - 4y' + 4y = 0$
$\lambda^2 - 4\lambda + 4 = 0 \;\Rightarrow\; (\lambda-2)^2 = 0$
$\lambda = 2$ (double), $\Delta = 16-16 = 0$
$\boxed{y = (C_1 + C_2 x)e^{2x}}$
Δ<0
Racines complexes — $y'' + 4y' + 5y = 0$
$\lambda^2 + 4\lambda + 5 = 0$, $\Delta = 16-20 = -4$
$\lambda = \dfrac{-4 \pm 2i}{2} = -2 \pm i$, donc $\alpha=-2,\;\beta=1$
$\boxed{y = e^{-2x}(C_1\cos x + C_2\sin x)}$
EX-O2-1 Ordre 2 homogène
Résoudre : $y'' - 2y' - 3y = 0$
EX-O2-2 Ordre 2 + Cauchy
Résoudre avec $y(0)=1, y'(0)=4$ : $y'' - 6y' + 10y = 0$
$\Delta = 36-40 = -4$, donc $\lambda = \frac{6\pm 2i}{2} = 3\pm i$. Racines complexes : $\alpha=3, \beta=1$
$y' = e^{3x}[(3C_1+C_2)\cos x + (3C_2-C_1)\sin x]$. En $x=0$ : $3C_1+C_2 = 4$. Avec $C_1=1$ : $C_2=1$

EDO Ordre 2 — Avec Second Membre

Type D · Ordre 2 · Non homogène

THÉORIE

Solution Particulière — Méthode des coefficients indéterminés

Forme
$$ay'' + by' + cy = f(x)$$

Solution générale : $y = y_h + y_p$

où $y_h$ est la solution de l'équation homogène (méthode précédente) et $y_p$ est une solution particulière.

Méthode — 3 étapes
  1. Résoudre $ay_h'' + by_h' + cy_h = 0$ (section précédente)
  2. Trouver $y_p$ selon la forme de $f(x)$ (tableau ci-dessous)
  3. Substituer $y_p$ dans l'équation et identifier les coefficients

Tableau des formes de $y_p$ selon $f(x)$ :

$f(x)$Forme de $y_p$Condition
Constante $k$$A$Si $c\neq 0$
Polynôme $P_n(x)$Polynôme de degré $n$Si $c\neq 0$
$e^{ax}$$Ae^{ax}$Si $a$ n'est pas racine de caract.
$e^{ax}$ ($a$ = racine simple)$Axe^{ax}$Multiplier par $x$
$e^{ax}$ ($a$ = racine double)$Ax^2e^{ax}$Multiplier par $x^2$
$\sin(bx)$ ou $\cos(bx)$$A\cos(bx) + B\sin(bx)$Toujours les deux
⚠️ Règle de résonance : Si la forme "naturelle" de $y_p$ est déjà dans $y_h$, multiplier par $x$ (ou $x^2$ si racine double). C'est le cas classique au piège !
Exemple — Second membre polynomial
Résoudre : $y'' - 2y' - 3y = 2x + 1$
1
Solution homogène
$\lambda^2-2\lambda-3=0 \;\Rightarrow\; \Delta=16 \;\Rightarrow\; \lambda_1=-1, \lambda_2=3$
$y_h = C_1e^{-x} + C_2e^{3x}$
2
Forme de $y_p$ : $f(x)=2x+1$ → polynôme deg 1
Poser $y_p = ax + b$, donc $y_p' = a$, $y_p'' = 0$
3
Substituer et identifier
$0 - 2a - 3(ax+b) = 2x+1$
$-3ax + (-2a-3b) = 2x+1$
$-3a=2 \Rightarrow a=-\tfrac{2}{3}$ et $-2(-\tfrac{2}{3})-3b=1 \Rightarrow b=\tfrac{1}{9}$
$y_p = -\tfrac{2}{3}x + \tfrac{1}{9}$
4
Solution générale
$\boxed{y = C_1e^{-x} + C_2e^{3x} - \tfrac{2}{3}x + \tfrac{1}{9}}$
Exemple — Second membre exponentiel avec résonance
Résoudre : $y'' - 4y' + 4y = e^{2x}$
1
Homogène
$\lambda^2-4\lambda+4=0 \Rightarrow (\lambda-2)^2=0 \Rightarrow \lambda=2$ (double)
$y_h = (C_1+C_2x)e^{2x}$
2
Forme $y_p$ — RÉSONANCE !
$f(x)=e^{2x}$ et $\lambda=2$ est racine double
Donc poser $y_p = Ax^2e^{2x}$ (multiplier par $x^2$)
3
Calculer $y_p'$ et $y_p''$, substituer
Après calcul et simplification : $2Ae^{2x} = e^{2x} \Rightarrow A = \tfrac{1}{2}$
$y_p = \tfrac{1}{2}x^2e^{2x}$
4
Solution générale
$\boxed{y = (C_1+C_2x)e^{2x} + \tfrac{1}{2}x^2e^{2x}}$
EX-PART-1 O2 avec 2nd membre
Résoudre : $y'' + 9y = 9e^{3x}$
$\lambda^2+9=0 \Rightarrow \lambda = \pm 3i$. Racines complexes : $\alpha=0$, $\beta=3$
$y_p=Ce^{3x}$, $y_p''=9Ce^{3x}$. Substituer : $9Ce^{3x}+9Ce^{3x}=9e^{3x} \Rightarrow 18C=9 \Rightarrow C=1/2$

Exercice 1 — EDO séparable (Syllabus 1.8d)

~25 pts

Résoudre avec $x_0=5, y_0=-1$ : $(x-4)xy' + y = 0$

Étape 1 — Mise en forme séparable
$\dfrac{dy}{y} = -\dfrac{dx}{x(x-4)}$
On utilise les fractions partielles : $\dfrac{1}{x(x-4)} = \dfrac{A}{x} + \dfrac{B}{x-4}$
$1 = A(x-4)+Bx \;\Rightarrow\; A=-\tfrac{1}{4},\; B=\tfrac{1}{4}$
Étape 2 — Intégration
$\ln|y| = \tfrac{1}{4}\ln|x| - \tfrac{1}{4}\ln|x-4| + C = \tfrac{1}{4}\ln\left|\dfrac{x}{x-4}\right| + C$
$y = K\left(\dfrac{x}{x-4}\right)^{1/4}$
Étape 3 — Condition initiale $y(5) = -1$
$-1 = K\left(\dfrac{5}{1}\right)^{1/4} = K\cdot 5^{1/4}$
$K = -5^{-1/4}$
$\boxed{y = -\left(\dfrac{x}{5(x-4)}\right)^{1/4}}$

Exercice 2 — EDO linéaire ordre 1 (Syllabus 1.14e)

~25 pts

Résoudre : $\dfrac{dr}{d\theta} + 3r = 2$

Étape 1 — Identifier le type et résoudre l'homogène
Forme standard avec $p(\theta)=3$, $q(\theta)=2$.
Homogène : $\dfrac{dr}{r} = -3\,d\theta \Rightarrow r_h = Ce^{-3\theta}$
Étape 2 — Solution particulière (variation de la constante)
Poser $r = C(\theta)e^{-3\theta}$, dériver et substituer :
$C'e^{-3\theta} - 3Ce^{-3\theta} + 3Ce^{-3\theta} = 2$
$C'e^{-3\theta} = 2 \;\Rightarrow\; C' = 2e^{3\theta} \;\Rightarrow\; C = \tfrac{2}{3}e^{3\theta}$
$r_p = \tfrac{2}{3}e^{3\theta}\cdot e^{-3\theta} = \tfrac{2}{3}$
Étape 3 — Solution générale
$\boxed{r(\theta) = Ce^{-3\theta} + \dfrac{2}{3}}$

Exercice 3 — Ordre 2 homogène + Cauchy (Syllabus 1.27b)

~25 pts

Résoudre avec $y(0)=1, y'(0)=4$ : $y'' - 6y' + 10y = 0$

Étape 1 — Équation caractéristique
$\lambda^2 - 6\lambda + 10 = 0$, $\Delta = 36-40 = -4$
$\lambda = \dfrac{6 \pm 2i}{2} = 3 \pm i$ → racines complexes : $\alpha=3, \beta=1$
Étape 2 — Solution générale et application des CI
$y = e^{3x}(C_1\cos x + C_2\sin x)$
$y(0)=C_1=1$
$y' = e^{3x}[(3C_1+C_2)\cos x + (3C_2-C_1)\sin x]$
$y'(0) = 3\cdot 1 + C_2 = 4 \;\Rightarrow\; C_2 = 1$
$\boxed{y = e^{3x}(\cos x + \sin x)}$

Exercice 4 — Ordre 2 avec 2nd membre trig (Syllabus 1.28j)

~25 pts

Résoudre : $y'' - 5y' + 6y = -3\sin x$

Étape 1 — Solution homogène
$\lambda^2-5\lambda+6=0 \Rightarrow (\lambda-2)(\lambda-3)=0$
$\lambda_1=2, \lambda_2=3$, $y_h = C_1e^{2x}+C_2e^{3x}$
Étape 2 — Solution particulière
$f(x)=-3\sin x$ → poser $y_p = A\cos x + B\sin x$
$y_p' = -A\sin x + B\cos x$, $y_p'' = -A\cos x - B\sin x$
Substituer : $(-A-B\cdot(-5)+6A)\cos x + (-B+A\cdot(-5)+6B)\sin x = -3\sin x$
— Coeff $\cos x$ : $5A-5B = 0 \Rightarrow A=B$
— Coeff $\sin x$ : $5A+5B = -3 \Rightarrow 10A=-3 \Rightarrow A=-\tfrac{3}{10}$
$y_p = -\tfrac{3}{10}(\cos x + \sin x)$
Étape 3 — Solution générale
$\boxed{y = C_1e^{2x}+C_2e^{3x} - \dfrac{3}{10}(\cos x + \sin x)}$

Exercice 5 — Ordre 2 Cauchy complet (Syllabus 1.29b)

~25 pts

Résoudre avec $y(0)=0, y'(0)=0$ : $y'' - 4y' + 4y = e^{2x} + x^2$

Étape 1 — Homogène
$(\lambda-2)^2=0 \Rightarrow \lambda=2$ double
$y_h = (C_1+C_2x)e^{2x}$
Étape 2 — $y_p$ pour $e^{2x}$ (résonance!)
$\lambda=2$ est racine double → poser $y_{p1} = ax^2e^{2x}$
Après calcul : $2a = 1 \Rightarrow a = \tfrac{1}{2}$, donc $y_{p1} = \tfrac{1}{2}x^2e^{2x}$
Étape 3 — $y_p$ pour $x^2$
Poser $y_{p2} = bx^2+cx+d$
$y_{p2}'' - 4y_{p2}' + 4y_{p2} = x^2$
$2b - 4(2bx+c) + 4(bx^2+cx+d) = x^2$
$4bx^2 + (-8b+4c)x + (2b-4c+4d) = x^2$
$b=\tfrac{1}{4}$, $c=\tfrac{1}{2}$, $d=\tfrac{3}{8}$
$y_{p2} = \tfrac{1}{4}x^2 + \tfrac{1}{2}x + \tfrac{3}{8}$
Étape 4 — Solution générale + CI
$y_g = (C_1+C_2x)e^{2x} + \tfrac{1}{2}x^2e^{2x} + \tfrac{1}{4}x^2 + \tfrac{1}{2}x + \tfrac{3}{8}$
$y(0)=0$: $C_1 + \tfrac{3}{8} = 0 \Rightarrow C_1 = -\tfrac{3}{8}$
$y'(0)=0$: $2C_1+C_2+\tfrac{1}{2} = 0 \Rightarrow C_2 = -\tfrac{1}{2}+\tfrac{3}{4} = \tfrac{1}{4}$
$\boxed{y = (-\tfrac{3}{8}+\tfrac{1}{4}x)e^{2x} + \tfrac{1}{2}x^2e^{2x} + \tfrac{1}{4}x^2 + \tfrac{1}{2}x + \tfrac{3}{8}}$
