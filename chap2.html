DÉFINITION
Valeur propre & Vecteur propre

Soit \(M \in \text{Mat}(n\times n, \mathbb{R})\). Un scalaire \(\lambda\) est une valeur propre de \(M\) s'il existe un vecteur non nul \(\vec{v}\) tel que :

\[ M\vec{v} = \lambda\vec{v} \]

Le vecteur \(\vec{v}\) est alors un vecteur propre associé à \(\lambda\).

Le vecteur nul n'est JAMAIS un vecteur propre par définition.
EXAM ★★★
Polynôme caractéristique & Spectre
\[ p_M(\lambda) = \det(M - \lambda I) \]

Les valeurs propres de \(M\) sont exactement les racines de \(p_M(\lambda)\).

Le spectre de \(M\) = l'ensemble des valeurs propres : \(\text{Sp}(M) = \{\lambda_1, \lambda_2, \ldots\}\)

Pour une matrice 3×3, le polynôme est de degré 3. Toujours chercher une racine évidente (1, −1, 2, 3…) puis diviser.
SOUS-ESPACE PROPRE
\[ V_\lambda = \ker(M - \lambda I) \]

C'est le sous-espace vectoriel des vecteurs propres associés à \(\lambda\) (+ le vecteur nul).

Pour le calculer : résoudre \((M - \lambda I)\vec{v} = \vec{0}\) par Gauss.

MULTIPLICITÉ

Algébrique : multiplicité de \(\lambda\) comme racine de \(p_M\).

Géométrique : \(\dim(V_\lambda) = \dim\ker(M-\lambda I)\)

Toujours : \(\dim(V_\lambda) \leq \text{mult. algébrique}\)

EXAM ★★★ — LE PIPELINE COMPLET
Diagonalisation
  • Calculer \(p_M(\lambda) = \det(M - \lambda I)\)
  • Trouver les racines → spectre \(\{\lambda_1, \lambda_2, \lambda_3\}\)
  • Pour chaque \(\lambda_i\) : résoudre \((M-\lambda_i I)\vec{v}=\vec{0}\), extraire une base de \(V_{\lambda_i}\)
  • Former \(B\) dont les colonnes sont les vecteurs propres trouvés
  • \(D = B^{-1}MB = \text{diag}(\lambda_1, \lambda_2, \lambda_3)\) est diagonale
  • Pour \(M^n\) : \(M^n = BD^nB^{-1}\) avec \(D^n = \text{diag}(\lambda_1^n, \lambda_2^n, \lambda_3^n)\)
CONDITION DE DIAGONALISABILITÉ

Une matrice \(n\times n\) est diagonalisable si et seulement si :

\[\sum_i \dim(V_{\lambda_i}) = n\]

Autrement dit : on peut former une base de vecteurs propres.

MATRICES SEMBLABLES

Si \(D = B^{-1}MB\), alors \(M\) et \(D\) sont semblables. Elles ont :

  • Même polynôme caract.
  • Même spectre
  • Même déterminant
  • Même trace
ASTUCE EXAM
Calculer M³ (ou Mⁿ)

Comme \(M = BDB^{-1}\) :

\[M^3 = (BDB^{-1})(BDB^{-1})(BDB^{-1}) = BD^3B^{-1}\]
\[D^3 = \begin{pmatrix}\lambda_1^3 & 0 & 0\\ 0 & \lambda_2^3 & 0\\ 0 & 0 & \lambda_3^3\end{pmatrix}\]
→ Exam Août 2025 : \(\lambda_1=1, \lambda_2=2, \lambda_3=3\) donc \(D^3 = \text{diag}(1,8,27)\)
ERREURS CLASSIQUES
  • Oublier de vérifier que les vecteurs propres forment une base (sinon M n'est pas diagonalisable)
  • Confondre \(B^{-1}MB\) avec \(BMB^{-1}\) — l'ordre compte !
  • Ne pas normaliser : les vecteurs propres peuvent être quelconques (pas besoin d'orthonormaliser sauf si demandé)
  • Calculer le déterminant par la règle de Sarrus pour les 3×3 (plus rapide)

THÉORÈME — Matrices Semblables
Si \(M\) et \(D\) sont semblables, elles ont le même polynôme caractéristique

Énoncé : Si \(D = B^{-1}MB\), alors \(p_D(\lambda) = p_M(\lambda)\).

\[\begin{align} p_D(\lambda) &= \det(D - \lambda I)\\ &= \det(B^{-1}MB - \lambda I)\\ &= \det(B^{-1}MB - \lambda B^{-1}IB)\\ &= \det(B^{-1}(M - \lambda I)B)\\ &= \det(B^{-1})\det(M-\lambda I)\det(B)\\ &= \frac{1}{\det B}\det(M-\lambda I)\det(B)\\ &= \det(M - \lambda I) = p_M(\lambda) \end{align}\]

✓ On a utilisé : \(\det(ABC) = \det(A)\det(B)\det(C)\) et \(\det(B^{-1}) = \frac{1}{\det(B)}\)

Corollaires importants :

  • Même spectre → valeurs propres sont des invariants de similarité
  • Même déterminant (\(= (-1)^n \cdot \text{terme constant du poly. caract.}\))
  • Même trace (\(= \text{somme des valeurs propres} = \text{somme des diagonaux}\))
  • Même rang
THÉORÈME — Vecteurs propres indépendants
Des vecteurs propres associés à des valeurs propres distinctes sont linéairement indépendants

Conséquence directe : Si \(M \in \text{Mat}(n\times n)\) a \(n\) valeurs propres distinctes, alors \(M\) est diagonalisable.

\[\text{Sp}(M) = \{\lambda_1 \neq \lambda_2 \neq \cdots \neq \lambda_n\} \implies M \text{ diagonalisable}\]

Soient \(\vec{v}_1, \vec{v}_2\) vecteurs propres pour \(\lambda_1 \neq \lambda_2\). Supposons \(\alpha\vec{v}_1 + \beta\vec{v}_2 = \vec{0}\).

\[M(\alpha\vec{v}_1 + \beta\vec{v}_2) = \alpha\lambda_1\vec{v}_1 + \beta\lambda_2\vec{v}_2 = \vec{0}\]
\[\lambda_1(\alpha\vec{v}_1 + \beta\vec{v}_2) = \alpha\lambda_1\vec{v}_1 + \beta\lambda_1\vec{v}_2 = \vec{0}\]

En soustrayant : \(\beta(\lambda_2 - \lambda_1)\vec{v}_2 = \vec{0}\). Puisque \(\lambda_2 \neq \lambda_1\) et \(\vec{v}_2 \neq \vec{0}\), on a \(\beta = 0\). Puis \(\alpha = 0\). ✓

FORMULE CLÉ — Puissance de matrice
Calculer \(M^k\) via la diagonalisation
\[M = BDB^{-1} \implies M^k = BD^kB^{-1}\]

Preuve :

\[M^k = \underbrace{(BDB^{-1})(BDB^{-1})\cdots(BDB^{-1})}_{k\text{ fois}} = BD\underbrace{(B^{-1}B)}_{=I}D\cdots DB^{-1} = BD^kB^{-1}\]
\[D^k = \begin{pmatrix}\lambda_1^k & & \\ & \ddots & \\ & & \lambda_n^k\end{pmatrix}\]
Exam : on ne diagonalise pas M "pour le plaisir" — c'est toujours pour calculer M^n ensuite !
